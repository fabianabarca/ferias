Sitio en desarrollo

Datos abiertos

Los datos en deferia.cr son abiertos y están disponibles para su uso y reutilización. Fueron recopilados por estudiantes de trabajo comunal universitario desde el Consejo Nacional de Producción, la Junta Nacional de Ferias del Agricultor, municipalidades, formularios para el público en general y visitas a las mismas, entre otras fuentes.

Sobre las ferias

Los datos de las ferias incluyen información sobre su ubicación, horarios de apertura, servicios y otros. La información de las ferias es revisada periódicamente.

Excel CSV

API

deferia.cr ofrece un RESTful API para acceder a los datos de las ferias.

GET deferia.cr/datos/api/ferias
Lista de todas las ferias en la base de datos.
GET deferia.cr/datos/api/ferias/{feria_url}
Información de una feria específica.
